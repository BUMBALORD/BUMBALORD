Ruby: Strings & Integers

  • String examples: "Hello World!", "12345", "Ruby is awesome!"
  • Integer examples: 1, 2, 3, 100, 1000000
  • String Methods:
    • "hello".upcase ===> "HELLO"
    • "HELLO".downcase ===> "hello"
    • "hello".capitalize ===> "Hello"
  • Integer Methods:
    • 1.odd? ===> true
    • 2.odd? ===> false
    • 14.even? ===> true

Arrays & Hashes

  • Array: []
  • Hash: {}
  • Size Method:
    • [1, 2, 3, 4].size ===> 4
    • { one: 1, two: 2, three: 3, four: 4 }.size ===> 4
  • Has/Include Method:
    • [1, 2, 3, 4].include?(4) ===> true
    • { one: 1, two: 2, three: 3, four: 4 }.has_key?(:one) ===> true
  • Fetching Values/Elements:
    • Fetching Array Elements:
    • Students = ['David', 'Emily']
    • Students[0] ===> 'David'
    • Fetching Hash Values:
    • Students = { 1 => 'David', 2 => 'Emily' }
    • Students[1] ===> 'David'
