Enumerable Method - .count

Feb 6th, 2015

You've been late with my methods 3 times! I've been counting!"


For this technical blog, I have chosen to focus on the method "count." Ruby-docs defines count as follows:


"Returns the number of items in enum through enumeration. If an argument is given, the number of items in enum that are equal to item are counted. If a block is given, it counts the number of elements yielding a true value."


This means that "count" can enumerate over what we tell it to, and give us back a handful of different answers based on what we tell it to do. For example, let's say we have a random array of numbers:


my_array = [1,2,3,4,4,4,5,6,7,8,8,9,10]


If we were to call "count" on our array directly, we would recieve the number of seperate integers within our array, like so:


my_array.count

#=> would return us 13, as there are 13 seperate integers within the array.


Simple, right? However, we can use "count" to get a lot more info than that out of our sample array. We can feed count an argument, and it will seek out that argument within the array. By seek out, I mean that count can go through each integer within the array and identify the one that we are specifically looking for, and tell us how many times it occurs. For example:


my_array.count(1)

#=> would return us 1, as there is only a single 1 in the array

my_array.count(4)

#=> would return us 3, as there are 3 seperate 4's in the array.


We can also feed count a block of code and have it iterate over the array with whatever we feed it. This can be used for getting more complex data out of our array, instead of writing many more lines of code. If we wanted to know how many of our integers within our array was divisible by 2, we would simply write:


my_array.count { |x| x%2 == 0}

#=> would return 8, as there are 8 separate integers within our array that can divided by 2 evenly.


These are just a few ways that we can use "count" as an enumerable method. We could feed it multiple arguments, or have it do any range of arithmatic within the block of code we feed it. While simple to use, "count" is a very powerful method that has many uses in Ruby.
